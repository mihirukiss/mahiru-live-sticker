# B站直播间弹幕表情选择器

在直播间弹幕输入框上方添加一个表情图标, 鼠标移动上去后会浮动显示一个弹幕表情选择框, 点击表情即可快速输入表情代码到弹幕输入框

目前支持[真绯瑠Mahiru](https://live.bilibili.com/21672024)与[恋诗夜Koxia](https://live.bilibili.com/23260932)的直播间
